# Higher or Lower Game

A fun and interactive game where you guess which celebrity has more Instagram followers! You earn points by choosing the celebrity with the higher follower count.

## Features
- Randomized selection of two celebrities for each round.
- User input to select which celebrity has more followers.
- Score tracking to see how well you do.

## How to Play
1. The game will present two celebrities, each with their description and country.
2. You need to guess which celebrity has more Instagram followers by typing 'A' or 'B'.
3. Your score will be updated with each correct guess. If you guess wrong, the game will end and show your final score.

## Prerequisites
- Python 3.x
- `game_data` module (or any data source with celebrity details)
- `art` module (for any ASCII art used)
